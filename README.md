# Snake Web Game

이 프로젝트는 HTML5와 JavaScript로 만든 간단한 웹 기반 스네이크(Snake) 게임입니다.

## 특징
- 난이도(Easy/Hard) 선택 가능
- 속도(느림/보통/빠름) 조절 가능
- 점수 및 랭킹 저장 (브라우저 LocalStorage 활용)
- 키보드 방향키로 조작
- GitHub Pages를 통해 웹에서 바로 플레이 가능

## 실행 방법
1. 이 저장소를 클론하거나, [GitHub Pages](https://hjhjhj0000-boop.github.io/snake-web/)에서 바로 실행할 수 있습니다.
2. `index.html`을 브라우저에서 열면 게임을 플레이할 수 있습니다.

## GitHub Pages 배포

## 조작법
- 방향키(←, ↑, →, ↓): 뱀 이동
- Space Bar: 일시정지/계속
- 각종 버튼: 난이도/속도/재시작 등 조절
